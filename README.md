# SQL-Task-Solutions

This repository contains my SQL practice exercises and their solutions, created to strengthen and showcase my portfolio.

The tasks are primarily based on the **Oracle Database Sample Schemas 23c** and **SQL Server AdventureWorks** databases. Some exercises are also adapted from projects and lessons completed on my [DataCamp portfolio](https://www.datacamp.com/portfolio/nigarmajidzada).

You can explore the Oracle database structure here:
[Oracle Database Sample Schemas Diagrams](https://docs.oracle.com/en/database/oracle/oracle-database/26/comsc/schema-diagrams.html#GUID-D268A4DE-BA8D-428E-B47F-80519DC6EE6E).
You can explore the SQL Server AdventureWorks structure here: https://github.com/microsoft/sql-server-samples/tree/master/samples/databases/adventure-works

Each task is saved as a separate `.sql` file, including a detailed **problem description** written as comments within the file.
